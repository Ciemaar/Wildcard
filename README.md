# Wildcard Management System (WMS)

A web application to manage missions for "The Wildcard Project"—an interactive photography game.
The system allows users to manage a database of text missions, categorize them, and generate print-ready PDFs containing these missions formatted onto physical cards.

## Quickstart
See `RUNBOOK.md` for running instructions.

## Core Purpose
Provide a complete system for managing photography game missions and generating print-ready cards with crop marks.

## Upcoming Features
- **User Authentication:** Support for individual user accounts and personal mission catalogs.
- **Custom Print Layouts:** Ability to select different grid layouts and paper sizes (e.g., A4 vs US Letter).
- **Advanced Filtering:** Enhanced dashboard tools to search and filter missions by tags and custom fields.
