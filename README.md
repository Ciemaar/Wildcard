# Wildcard Management System (WMS)

A web application to manage prompts for "The Wildcard Project"—an interactive photography game.
The system allows users to manage a database of text prompts, categorize them, and generate print-ready PDFs containing these prompts formatted onto physical cards.

## Quickstart
See `RUNBOOK.md` for running instructions.

## Core Purpose
Provide a complete system for managing photography game prompts and generating print-ready cards with crop marks.
